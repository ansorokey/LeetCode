Declaring Variables:
```py
a = 7
a = 8
b = "Reyes"
x = y = z = 0
i, j = 0, 0
```

Printing:
```py
print(a, b, c)
```

Iteration:
```py
# Ranges
for i in range(x):
    pass

for i in range(len(a)):
    pass

for i in range(start, stop, step):
    pass

# Items
for c in string:
    pass
```
Conditionals
```py
if a or b:
    pass

if a and b:
    pass

if not a:
    pass
```
